# Smart Attendance System

A Face Recognition-based Attendance System using OpenCV and MySQL.

##  Features
- Real-time face detection
- Face recognition using the LBPH algorithm
- Automatic attendance marking
- Duplicate attendance prevention

## Technologies Used
- Python
- OpenCV
- MySQL

##  How to Run
1. Run capture.py to collect data
2. Run train.py to train the model
3. Run main.py to start the system

##  Output
- Detects face
- Recognizes user
- Stores attendance in MySQL database
